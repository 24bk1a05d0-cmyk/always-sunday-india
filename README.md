# Always Sunday India - Premium Café Website ☕

A highly animated, visually immersive, and premium café website that feels like a digital escape experience.

## 🎨 Features

### Core Experience
- **Life's a Holiday Theme** - Every scroll feels like entering a relaxed Sunday vibe
- **Playful + Premium + Instagram Aesthetic** - Blend of modern design with fun interactions
- **Vibrant Color Palette** - Sunset orange, warm yellow, soft pastel pinks, creamy neutrals

### 🎭 Mood Selector (Unique Feature!)
Users can select their vibe:
- **Chill 😌** - Relaxed browsing experience
- **Work 💻** - Productive, focused atmosphere
- **Date ❤️** - Romantic ambiance
- **Party 🎉** - Celebratory energy

The website dynamically changes:
- Background images
- Color themes
- Visual atmosphere

### ⚡ Animations & Interactions

#### Smooth Animations
- Parallax scrolling effects
- Floating elements (coffee cups, smiley icons, doodles, emojis)
- Scroll-triggered reveals with fade + slide effects
- Smooth page transitions

#### Hover Effects
- Buttons expand with glow effects
- Menu items pop with scale animations
- Image zoom on hover
- Interactive cards with animated highlights

#### Custom Cursor
- Coffee cup icon that follows mouse
- Changes on hover over interactive elements
- Smooth spring animation

### 🏠 Sections

1. **Hero Section**
   - Full-screen immersive experience
   - Dynamic background based on mood
   - Bold tagline: "Life's a holiday"
   - Animated scroll indicator

2. **Experience Section**
   - Split-screen scroll storytelling
   - Rotating image carousel
   - Text animations synchronized with images
   - Features: Specialty Coffee, Fusion Food, Familiar Faces, Cozy Interiors

3. **Menu Section**
   - Interactive tabbed navigation (Coffee, Matcha & Benne, Mains, Desserts)
   - Hover previews with image pop-ups
   - Tags: "Instagram Favorite", "Must Try", mood-specific tags
   - Glassmorphism cards

4. **Instagram Gallery**
   - Masonry grid layout (Pinterest-style)
   - Hover zoom effects with captions
   - Lightbox modal for full-size viewing
   - Social engagement metrics

5. **Vibes Section**
   - Animated cards for different occasions:
     - Brunch with Friends
     - Date Vibes
     - Work from Café
     - Birthday Hangouts
   - Glow effects on hover
   - Dynamic background overlays

6. **Visit Us Section**
   - Google Maps embed
   - Animated info cards with icons:
     - Location (Khajaguda, Hyderabad)
     - Timings
     - Parking
     - Pet-friendly 🐶
     - Free WiFi
     - Contact
   - "Get Directions" CTA

7. **CTA Section**
   - Bold conversion message: "Not just a café. It's a vibe."
   - WhatsApp booking integration
   - Order now links (Zomato/Swiggy)
   - Social media links
   - Animated gradient background

### 🎯 Additional Features

- **Floating Elements** - Emojis and icons that float across the screen
- **Glassmorphism Design** - Frosted glass effects throughout
- **Responsive Design** - Works perfectly on mobile and desktop
- **Custom Scrollbar** - Branded with café colors
- **Sticky Navbar** - Animated, transparent to solid on scroll
- **Scroll to Top Button** - Quick navigation
- **Mood Indicator** - Shows active mood in corner
- **Welcome Animation** - Branded splash screen on load

## 🛠️ Tech Stack

- **React** - UI framework
- **Motion (Framer Motion)** - Smooth animations
- **Tailwind CSS v4** - Styling with custom theme
- **React Responsive Masonry** - Instagram gallery layout
- **Lucide React** - Beautiful icons
- **Google Fonts (Poppins)** - Premium typography
